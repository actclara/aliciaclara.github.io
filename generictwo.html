Investigating Correlation In Movies

What factors contribute to movies getting higher gross earnings?

Home

About the Data

This movies dataset is publicly available in kaggle. There are 6820 movies in the dataset (220 movies per year, 1986-2016). Each movie has the following attributes: budget,production company, country of origin, director, movie genre, rating (R,PG,etc.), release date (YYYY-MM-DD)) , runtime, IMDb user rating, number of user votes, main actor/actress, write and year of release. All these variables are cleaned and analyzed to see whether one or more of them have any strong association with gross earnings.

Code Script

Using several Pandas libraries such as pandas, numpy, seaborn and matplotlib to clean, analyzed, and visualize data. The correlation matrix will show a value ranging from 0.00 to 1.00. The higher the value, the greater the correlation or the association between the variables.

Github

Conclusion

After comparing all variables against the gross earnings, it appears votes (corr = 0.6) and budget (corr = 0.75) have the greatest correlation strength to gross earnings. In summary, when a movie has a high number of user votes and huge budget, it is likely to have high gross earnings compared to movies with low budget and low number of user votes.
